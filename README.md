# School Management System Business Analysis Project

## Project Overview
This project analyzes the requirements for a School Management System designed to streamline student registration, attendance tracking, fee processing, result management, and academic reporting.

## Business Problem
Manual school administration processes result in inefficiencies, data inaccuracies, and delayed reporting.

## Project Objectives
- Digitize student registration
- Improve attendance tracking
- Simplify fee management
- Generate academic reports

## My Role
Business Analyst

## Deliverables
- Business Case
- Business Requirements Document (BRD)
- Stakeholder Analysis
- Epics, Feature lists and User Stories
- Use Cases
- Process Flow Diagrams
- Requirements Traceability Matrix (RTM)
- Data Model-ERD
- Meeting Notes

## Tools Used
- Microsoft Excel
- PDF
- Visio
- GitHub

## Key Features
- Student Registration
- Attendance Management
- Fee Management
- Result Management
  

## Project Outcome
A comprehensive requirements package for the development of a School Management System.
